## ¿Como funcionan exactamente?

EAS (Expo Application Services) es el sistema de build y deploy en la nube de Expo. Te permite:

- Construir versiones para iOS y Android con eas build
- Enviar tu app a las tiendas con eas submit
- Hacer updates OTA (Over The Air) con eas update

Con estas herramientas puedes olvidarte de los entornos locales de build y liberar versiones mucho más rápido.
